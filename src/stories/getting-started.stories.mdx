# Getting Started

**Coborns Web is the front-end library of React components and sample layouts with associated Storybook stories.**

## Repository Access

The `coborns-web` repository is located on Bitbucket here: [https://bitbucket.org/bswing/coborns-web/](https://bitbucket.org/bswing/coborns-web/)

The repository is private and access must be granted. If you wish to have access to the repository please email either Dylan McDonald at [dmcdonald@bswing.com](mailto:dmcdonald@bswing.com) or Eric Freeberg at [efreeberg@bswing.com](mailto:efreeberg@bswing.com).

## Running Locally

After cloning the repository locally you will then need to install the NPM dependencies.

1. Install dependencies using npm ```npm install```
2. After dependencies install successfully run storybook using the script ```npm run storybook```

**Notes**
- Be sure you have a fairly new version of [Node.JS](https://nodejs.org/en/) installed, ideally LTS. There is no specific version needed as of right now. bswing developers currently running `node v14.17.3` and `npm v7.20.1`. Errors will occur if you are running an unsupported version.
- If using Visual Studio Code, here are the recommended extensions for QoL improvements within the editor: [ESLint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint), [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode), [Tailwind CSS IntelliSense](https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss)

## Third-Party Libraries

- [React](https://reactjs.org/) - React is the front-end library chosen for the new Coborns Web work. All components and sample layouts are built using React.
- [Storybook](https://storybook.js.org/) - Storybook is the tool used to showcase components and sample layouts for development and testing.
- [Tailwind](https://tailwindcss.com/) - Tailwind is the core utility-first CSS framework that Coborns Web is built upon.
- [Headless UI](https://headlessui.dev/) - Headless UI is a fully accessible UI component library that integrates with Tailwind.
- [Heroicons](https://heroicons.com/) - Heroicons is the icon library used in the new Coborns Web work.
